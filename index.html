Roulette Game

Game Rules:---

  • Player can bet on any number between (0-35). if the betting number and outcome matches then player wins 35X the betting amount.
  • Winning message along with win amount is displayed OR a message "Better luck next time" is displayed.
  • User should be asked to rebet, if yes then user should be asked to place bet amount and number. Loop continues till player quits Or have lost all the money
